# Rect

## 概述

如果rects是空指针nullptr，默认Buffer大小为脏区。

**相关模块：** [NativeWindow](capi-nativewindow.md)

**所在头文件：** [external_window.h](capi-external-window-h.md)

## 汇总

### 成员变量

| 名称       | 描述              |
| ---------- | ----------------- |
| int32_t x  | 矩形框起始x坐标。 |
| int32_t y  | 矩形框起始y坐标。 |
| uint32_t w | 矩形框宽度。      |
| uint32_t h | 矩形框高度。      |
