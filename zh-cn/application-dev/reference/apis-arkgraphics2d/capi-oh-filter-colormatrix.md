# OH_Filter_ColorMatrix

## 概述

定义一个用来创建滤镜效果的矩阵。

**起始版本：** 12

**相关模块：** [effectKit](capi-effectkit.md)

**所在头文件：** [effect_types.h](capi-effect-types-h.md)

## 汇总

### 成员变量

| 名称          | 描述                               |
| ------------- | ---------------------------------- |
| float val[20] | 自定义颜色矩阵，值是一个54的数组。 |
