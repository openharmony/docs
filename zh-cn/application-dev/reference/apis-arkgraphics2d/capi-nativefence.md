# NativeFence

## 概述

提供NativeFence功能，通过提供的接口，可以实现对fenceFd阻塞指定时间、永久阻塞、关闭和检查fenceFd是否有效等操作。

**系统能力：** SystemCapability.Graphic.Graphic2D.NativeFence

**起始版本：** 20
