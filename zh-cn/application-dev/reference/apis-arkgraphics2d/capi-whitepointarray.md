# WhitePointArray

## 概述

提供白点数组结构体，白点是指在当前色域中表示白色的坐标。

**起始版本：** 13

**相关模块：** [NativeColorSpaceManager](capi-nativecolorspacemanager.md)

**所在头文件：** [native_color_space_manager.h](capi-native-color-space-manager-h.md)

## 汇总

### 成员变量

| 名称         | 描述               |
| ------------ | ------------------ |
| float arr[2] | 表示白点返回数组。 |
