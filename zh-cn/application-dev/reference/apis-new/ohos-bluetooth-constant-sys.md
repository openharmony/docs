# @ohos.bluetooth.constant    
constant模块提供了蓝牙中常量的定义。  
> **说明**   
>本模块首批接口从API version 10开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本  
  
## 导入模块  
  
```js    
import constant from '@ohos.bluetooth.constant'    
```
