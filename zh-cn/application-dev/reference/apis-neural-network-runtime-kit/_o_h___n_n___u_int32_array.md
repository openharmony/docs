# OH_NN_UInt32Array


## 概述

该结构体用于存储32位无符号整型数组。

**起始版本：** 9

**相关模块：**[NeuralNeworkRuntime](_neural_network_runtime.md)


## 汇总


### 成员变量

| 名称 | 描述 | 
| -------- | -------- |
| uint32_t \* [data](#data) | 无符号整型数组的指针 | 
| uint32_t [size](#size) | 数组长度 |
