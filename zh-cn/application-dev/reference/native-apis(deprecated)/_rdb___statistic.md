# Rdb_Statistic


## 概述

描述数据库表的端云同步过程的统计信息。

**起始版本：** 11

**相关模块：**[RDB](_r_d_b.md)


## 汇总


### 成员变量

| 名称 | 描述 |
| -------- | -------- |
| [total](_r_d_b.md#total) | 表示数据库表中需要端云同步的总行数。 |
| [successful](_r_d_b.md#successful) | 表示数据库表中端云同步成功的行数。 |
| [failed](_r_d_b.md#failed) | 表示数据库表中端云同步失败的行数。 |
| [remained](_r_d_b.md#remained) | 表示数据库表中端云同步剩余未执行的行数。 |
