# Camera_Rect

## 概述

矩形定义。

**起始版本：** 11

**相关模块：** [OH_Camera](capi-oh-camera.md)

**所在头文件：** [camera.h](capi-camera-h.md)

## 汇总

### 成员变量

| 名称 | 描述 |
| -- | -- |
| int32_t topLeftX | 左上角的X坐标。 |
| int32_t topLeftY | 左上角的Y坐标。 |
| int32_t width | 矩形宽度，单位为像素。 |
| int32_t height | 矩形高度，单位为像素。 |
