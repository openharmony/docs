# @ohos.multimodalInput.keyCode (键值)

按键设备的键码值，按键设备包括键盘、电源键、拍照键等。

> **说明：**
> 
> 本模块首批接口从API version 9开始支持。后续版本的新增接口，采用上角标单独标记接口的起始版本。

## 导入模块

```js
import { KeyCode } from '@kit.InputKit';
```
