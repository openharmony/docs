# IRect


## 概述

定义矩形框信息。

**起始版本：** 3.2

**相关模块：**[Display](_display_v10.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [x](#x) | 矩形框起始x坐标  | 
| int [y](#y) | 矩形框起始y坐标  | 
| int [w](#w) | 矩形框宽度  | 
| int [h](#h) | 矩形框高度  |
