# RGBColor


## 概述

定义灯的RGB模式。

**起始版本：** 3.2

**相关模块：**[Light](_light.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| int [brightness](#brightness) | 亮度值，范围为0-255。  | 
| int [r](#r) | 红色值，范围为0-255。  | 
| int [g](#g) | 绿色值，范围为0-255。  | 
| int [b](#b) | 蓝色值，范围为0-255。  |
