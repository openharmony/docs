# EffectInfo


## 概述

定义effect加载的音效信息。

**起始版本：** 4.0

**相关模块：**[HdiEffect](_hdi_effect.md)


## 汇总


### Public 属性

| 名称 | 描述 | 
| -------- | -------- |
| String [libName](#libname) | 指定用于创建控制器的音效库名称 | 
| String [effectId](#effectid) | 音效ID | 
| int [ioDirection](#iodirection) | 确定效果的方向 |
